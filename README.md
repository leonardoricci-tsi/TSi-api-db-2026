# TSi API com SQLite 🚀

Este projeto é uma API REST desenvolvida como parte da disciplina de Tecnologia em Sistemas para Internet (TSi). A aplicação demonstra a integração de um servidor Node.js com o banco de dados SQLite para operações de CRUD (Create, Read, Update, Delete).

## 🛠️ Tecnologias Utilizadas

* **Node.js**: Ambiente de execução Javascript server-side.
* **Express**: Framework web para criação de rotas e middlewares.
* **SQLite3**: Banco de dados relacional leve, armazenado em arquivo local.
* **JavaScript (ES6+)**: Utilizando módulos (`import/export`).

## ⚙️ Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
* [Node.js](https://nodejs.org/en/) (versão 16 ou superior)
* Um cliente HTTP (como Postman, Insomnia ou a extensão REST Client do VS Code)
